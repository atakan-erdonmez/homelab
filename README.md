# Homelab

Evolving homelab environment where I design, build, and operate systems with a focus on automation, infrastructure as code, and reliable, secure architecture.

## Structure

- `infrastructure/` → core system components and networking
- `services/` → self-hosted applications and workloads
- `ansible/` → configuration management and automation
- `tools/` → operational scripts and utilities

## Notes

This repository is used as a working environment. Sensitive data is excluded, and example configurations are provided where relevant.
